# SWU_Net_Login
> 西南大学（SWU）校园网锐捷认证命令行登录工具\
> A CLI Tool for Ruijie Campus Network Authentication (SWU)

------------------------------------------------------------------------

## 📌 项目简介 \| Introduction

`SWU_Net_Login` 是一个基于 Python 的命令行校园网登录工具，\
主要用于
**在西南大学校园网环境下，通过命令行方式直接连接校园网，不需要浏览器，多用于实验室、服务器等**

适用于服务器、远程实验环境、无 GUI 系统环境。

------------------------------------------------------------------------

## ✨ 功能特性 \| Features

-   命令行登录校园网
-   支持默认账号密码
-   支持手动输入
-   密码隐藏
-   自动获取 IP / MAC
-   适合服务器环境

------------------------------------------------------------------------

## 📥 安装方式 \| Installation

``` bash
git clone https://github.com/yourname/SWU_Net_Login.git
cd SWU_Net_Login
pip install -r requirements.txt
```

------------------------------------------------------------------------

## 🔧 配置说明 \| Configuration

``` bash
cp .env.example .env
```

填写账号密码：

``` env
RUIJIE_USERNAME=你的账号
RUIJIE_PASSWORD=你的密码
```

------------------------------------------------------------------------

## ▶️ 使用方法 \| Usage

``` bash
python -m ruijie_login.main
```
